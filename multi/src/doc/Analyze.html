Analyze TINY programs.
val analyze : (module Relational.Domain) -> int -> string -> string option -> unit
analyze domain descending input output analyzes RINY program in file input with abstract domain domain and outputs result as a commented program (or only invariants if Report.verbosity <= 1) in file output or standard output if no output is provided.

After fixpoint of loops are reached, performs descending descending iterations.
