Module type NonRelational.Domain

module type Domain = sig .. end
Module type of non relational domains.
val name : string
Name of the domain. Used to select it via command line arguments. Should be of the form [a-z][a-zA-Z0-9_]*.
val parse_param : string -> unit
parse_param s parses parameters in s. If s is of the form "dom:s'" where dom doesn't contain ':' and is not name, the parameter should be ignored. If dom is name, only s' should be considered (Utils.select_param name can be used for that). A warning should be issued for unrecognised parameters (use Utils.warn_unknown_param).
val fprint_help : Format.formatter -> unit
Outputs some help about the options recognized by parse_param.

Abstract Operators


val sem_itv : Q.t -> Q.t -> t
sem_itv n1 n2 returns an abstraction t of the interval [n1, n2]: 
[n1, n2] \subseteq \gamma(t).


Semantics of four basic arithmetic operations.
val sem_plus : t -> t -> t
val sem_minus : t -> t -> t
val sem_times : t -> t -> t
val sem_div : t -> t -> t
sem_plus t1 t2 returns a t' such that: 
{ n1 + n2 | n1 \in \gamma(t1), n2 \in \gamma(t2) } \subseteq \gamma(t').


Abstract operator for guard ">= 0".
val sem_geq0 : t -> t
sem_geq0 t returns a t' such that: 
{ n \in \gamma(t) | n >= 0 } \subseteq \gamma(t').

backsem_plus t1 t2 t returns a pair (t1', t2') such that: 
\forall n1 n2 : Z or R,
      (n1 \in \gamma(t1) /\ n2 \in \gamma(t2) /\ n1+n2 \in \gamma(t))
      -> (n1 \in \gamma(t1') /\ n2 \in \gamma(t2')).
