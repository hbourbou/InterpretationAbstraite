module type Domain = sig .. end
Module type for relational domains.
val name : string
Name of the domain. Used to select it via command line arguments. Should be of the form [a-z][a-zA-Z0-9_]*.
val parse_param : string -> unit
parse_param s parses parameters in s. If s is of the form "dom:s'" where dom doesn't contain ':' and is not name, the parameter should be ignored. If dom is name, only s' should be considered (Utils.select_param name can be used for that). A warning should be issued for unrecognised parameters (use Utils.warn_unknown_param).
val fprint_help : Format.formatter -> unit
Outputs some help about the options recognized by parse_param.

Abstract Operators



Abstract semantics of assignments and guards.
val assignment : Name.t -> Ast.expr -> t -> t
assignment n e t returns a t' such that: 
[|n = e;|](\gamma(t)) \subseteq \gamma(t').

val guard : Ast.expr * Ast.cmp -> t -> t
guard e t returns a t' such that: 
[|e >= 0|](\gamma(t)) \subseteq \gamma(t').
